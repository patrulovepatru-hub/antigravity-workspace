# Sistema Adaptativo para Freestylers (Adaptacion^3)

**Problema raíz**: Al improvisar pierdes el hilo - el cerebro va más rápido que la boca

## Los 3 puntos de fallo
1. **Delivery** - la ejecución física/rítmica
2. **Hilado** - conectar idea A con idea B con coherencia
3. **Enfoque** - mantener el norte de lo que querías decir

## La paradoja
> "Organizarse es clave, pero cada persona necesita un proceso diferente"

No puedes dar el mismo sistema a todos. El sistema debe **aprender cómo aprende** cada usuario.

## Concepto: Adaptación al cubo
```
Nivel 1: Sistema base (estructura general)
Nivel 2: Adapta al usuario (aprende sus patrones)
Nivel 3: Adapta la adaptación (ajusta CÓMO aprende de él)
```

## Posibles features
- [ ] Detección de patrones de pérdida de hilo (NLP sobre transcripciones)
- [ ] Feedback en tiempo real (¿wearable? ¿app con audio?)
- [ ] Mapa mental dinámico que se ajusta al estilo del usuario
- [ ] "Anclas" personalizadas - palabras/conceptos que lo recentran
- [ ] Histórico de sesiones para ver evolución

## Por investigar
- Speech-to-text en tiempo real (Whisper, Deepgram)
- Modelos de coherencia semántica
- Gamificación adaptativa
- Cómo medir "flow state" objetivamente
